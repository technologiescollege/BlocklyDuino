Advanced_map
==========

Re-map a number from a certain range to another.

Parameters
----------

| Param name | Description | Type     |
 ------------|-------------|----------
| value     | Number that is going to be re-mapped | `Number` |
| min1     | Lower limit of the original range | `Number` |
| max1     | Upper limit of the original range | `Number` |
| min2     | Lower limit of the new range | `Number` |
| max2     | Upper limit of the new range | `Number` |
